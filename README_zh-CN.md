OnColour
========

一个用重力感应器控制的游戏（在桌面电脑上用键盘控制，也可以用重力感应器——如果你的电脑足够高端大气上档次）

玩法
========

基本玩法
--------
左右倾斜设备，控制屏幕中央的小球进入大的圆并保持一段时间，即可获得该大圆的颜色。<br>
大圆的不透明度越高，可以获得的颜色越多。

不够爽？
--------
进入游戏“玩法”界面即可看到更多好玩之处。限于篇幅，此处不再作介绍。

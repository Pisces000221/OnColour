On Colour
========

一个用重力感应器控制的游戏（在桌面电脑上用键盘控制，也可以用重力感应器——如果你的电脑足够高端大气上档次）

游戏背景
========

小球从床上醒过来。<br>
他的小伙伴看到了他……“嗯……我觉得你貌似有点那个了……”<br>
“啊？真的？”他冲到镜子前，的确如此！原本白色的他现在正在一点一点地掉色！<br>
“这该怎么办呢？”他着急起来。要知道，如果构成他的颜色中红、绿、蓝三色中的任何一个降低到0，他就得“拜拜”了啊！<br>
颜料？那只会让他掉色更快。他掉的色是光线，不是颜料！<br>
LED灯泡？？没用……<br>
看来只有“光子”（一个圆形的东东，待在里面可以得到光的颜色）能够帮他了，而附近又没有……<br>
到底怎么办？？？？……<br>
……<br>
……<br>
于是，情急之下，他冲进了充满着“光子”的屏幕，期待着你能够帮助他解决这一问题……<br>
……<br>

玩法
========

基本玩法
--------
左右倾斜设备，控制屏幕中央的小球进入大的圆即“光子”并保持一段时间，即可获得该光子的颜色。<br>
光子的不透明度越高，可以获得的颜色越多。

不够爽？
--------
进入游戏“玩法”界面即可看到更多好玩之处。限于篇幅，此处不再作介绍。
